  • Simulink Veirification and Validationが必要 (R2010b以降)

Contents

  • Overview
  • How to run

Overview

(1) Select a model to be tested.

(2) Click Harness with expected output button

(3) A harness model will be created and expected outputs and model's outputs are comapred using Assertion block

(4) A signal builder block inside the harness model has a dummy inputs.

How to run

  1. Open cmtdtool_samplemode.mdl in cmtdtool's install directory.
  2. Click " Harness with expected output " button in cmtdtool GUI.
  3. Check that a new harness model is created.
  4. Run the following commands and you will see that the same operation can be done by API call .
open_system('cmtdtool_samplemode')
makeharness_compareresults('cmtdtool_samplemode');
